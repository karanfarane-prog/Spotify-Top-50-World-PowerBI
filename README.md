# 🎵 Spotify Top 50 World Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built using the Spotify Top 50 World dataset.

It provides insights into:

- Song popularity
- Artist performance
- Album type distribution
- Explicit vs Non-explicit songs
- Song duration analysis
- Popularity trends over time

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Excel/CSV

---

## 📊 Dashboard Features

- Interactive slicers
- KPI Cards
- Top 5 Songs
- Featured Song
- Popularity Distribution
- Album Type Analysis
- Artist Ranking
- Scatter Plot
- Time Series Analysis
